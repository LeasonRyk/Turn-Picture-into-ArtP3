# MakeCartoon

This repository contains a Python script to convert images into cartoon-style images using various image processing techniques.

## Features

- Convert any image to a cartoon-style image.
- Easy to use with a single command.

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Python 3.6 or higher
- Required Python packages (listed in `requirements.txt`)

## Installation

1. **Clone the repository**

    ```bash
    git clone https://github.com/yourusername/makecartoon.git
    cd makecartoon
    ```

2. **Create a virtual environment**

    ```bash
    python3 -m venv venv
    source venv/bin/activate  # On Windows use `venv\Scripts\activate`
    ```

3. **Install the dependencies**

    ```bash
    pip install -r requirements.txt
    ```

## Usage

To convert an image to a cartoon-style image, use the following command:

```bash
python makecartoon.py --image "PATH/TO/IMAGE.Type"
